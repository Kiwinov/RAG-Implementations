# RAG Implementations
This is a codeground for implementing, evaluating and testing features and RAG Components using a database of unsolved missing cases from a few Wikipedia pages.

## Contents
1. **Data:** Contains all the data extracted and cleaned, in both CSV and SQL formats.
2. **Vectorstores:** Contains all the embeddings of the data.
3. **RAG_Proj.ipynb** contains the primary code for the entire RAG Pipeline
